= POSTY ELEVEN =

== Description ==

This is a minimalist Wordpress theme based on Twenty Eleven and influenced by the Posterous.

== Features ==

* Removes the header image
* Removes the navigation bar
* Inherits most of the features from Twenty Eleven

== Frequently Asked Questions ==

= Why is there no navigation bar? =

It's a feature ;-)

The theme is designed to be minimalist. It does this by focusing on just the blogging.
